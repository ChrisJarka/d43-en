===== Return to Default Keyboard =====

Complete these steps to return to the default keyboard in the app:

  - Open the translationStudio app.
  - The **Edit ** window opens. Tap in the window as if to type - the keyboard slides up.
  - Tap-slide your finger on the keyboard's space bar to choose another language.
  - Select the default language.

Stop sliding the finger when the desired language shows.

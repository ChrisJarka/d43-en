====== Sample Door43 Code ======

==== Activity: ====

Code to display the activity of a user:

  * Replace "USER" with Door43 username
  * Count= sets number of results

<code>
**USER:**
{{changes>user=USER&count=12&render=pagelist(header,date,user,nocomments)&maxage=25920000}}
</code>

Code to display the activity in a namespace:

  * Replace "NS" with Door43 namespace
  * Count= sets number of results

<code>
**NAMESPACE:**
{{changes>ns=NS&count=12&render=pagelist(header,date,user,nocomments)&maxage=25920000}}
</code>


==== Tags: ====

Display pages with in namespace with specific tag:

<code>
{{topic>en:bible:notes:luk?draft&nocomments&simplelist}}
</code>

Display pages with in namespace with specific tag, but without another tag:

<code>
{{topic>en:bible:notes:luk?draft -cq&nocomments&simplelist}}
</code>

Display count of all tags in a namespace:

<code>
{{count>+&en:bible:team-info:training:quick-reference}}
</code>


==== Edit Page: ====

Open Door43 page in edit mode:

<code>
?do=edit&rev=0
</code>

==== Transclude: ====

Code to pull in content from a specific section:

<code>
{{section>:en:bible:notes:luk:01:01#important terms:&noheader}}
</code>

==== New-style full text search: ====

Find all draft tags in Romans NS:

<code>
{{pagequery>@en:bible:notes:rom {{tag>[^}]*draft[^}]*}}; fulltextregex; sort=title:asc; display={title}; natsort; layout=table;hidemsg;hidejump;bullet=square;fontsize=100%}}
</code>

Test finding KT pages with "drewc" tag and with "ktobs" tag:

<code>
{{pagequery>@en:obe:kt \{\{tag>(?=[^}]*drewc)(?=[^}]*ktobs)[^}]*\}\}; fulltextregex; sort=title:asc; display={title}; natsort; layout=table;hidemsg;hidejump;bullet=square;fontsize=100%}}
</code>


Test finding KT pages with "drewc" tag but without "ktobs" tag:

<code>
{{pagequery>@en:obe:kt \{\{tag>(?![^}]*ktobs)(?=[^}]*drewc)[^}]*\}\}; fulltextregex; sort=title:asc; display={title}; natsort; layout=table;hidemsg;hidejump;bullet=square;fontsize=100%}}
</code>

===== Old "page query" Text String: =====

Search ns for pages with or without text string:

<code>
{{pagequery>ns:en:bible:notes:mat "{{tag>create" -"{{tag>delete"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Search for inserted beginning **or**  ending <nowiki>HTML <font></nowiki> code in OBE pages:
<code>
{{pagequery>ns:en:obe "<font" | ns:en:obe "</font>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Search for inserted beginning **or**  ending <nowiki>HTML <font></nowiki> code in Bible notes pages:
<code>
{{pagequery>ns:en:bible:notes "<font" | ns:en:bible:notes "</font>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Find missing text portions:

<code>
{{pagequery>ns:en:bible:notes "Important Terms" -"===== Important Terms: ====="; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

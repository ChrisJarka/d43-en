====== Check Tags ======

==== Check on bad tags ====

This query checks the first tag to be one of the valid tags required to make the system work correctly: {create|delete|draft*|check|review|text|discuss|publish}. Any verses listed here are likely being overlooked above. Please include this query for every bible book admin page. (EQ)

{{pagequery>ns:en:bible:notes "{{tag>" -"{{tag>create" -"{{tag>delete" -"{{tag>draft" -"{{tag>draft1" -"{{tag>draft2" -"{{tag>check" -"{{tag>review" -"{{tag>text" -"{{tag>discuss" -"{{tag>publish"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}

====== Edit a Page ======

Only Door43 users can edit a page. Anyone can be a Door43 user. Go to [[Create an Account]] to create an account now.

Click on the 'login' link at the top right on any Door43 page. Enter your user ID and Password.

Go to a page to edit. Look for the Pencil Icon that is to the right of the content window.

Click the “Edit this page” icon (the pencil), the text editor window opens. 

Go to [[CKG (Graphical) Editor]] to learn how to use the graphical editor.\\
Go to [[Formatting Content]] to learn how to use the coding editor.\\
Go to [[Switching Editors]] to learn how to switch between the graphical and coding editors.\\

==== Check for Bad Tags ====

The following query checks the first tag to be one of the valid tags required to make the system work correctly: {create|draft(1-2)|review|discuss|publish}. Any terms listed here are not in the above chart.

{{pagequery>ns:en:obe "{{tag>" -"{{tag>delete" -"{{tag>create" -"{{tag>draft1" -"{{tag>draft2" -"{{tag>review" -"{{tag>discuss" -"{{tag>publish"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}

==== Check for Well Formed Tags ====

The following query checks that every page that has curly brace sets"{}" include the key word "tag>" in at least one set.

{{pagequery>ns:en:obe -ns:en:obe:kt:home -ns:en:obe:other:home -ns:en:obe:home "{" -"{{tag>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}

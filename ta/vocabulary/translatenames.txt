===== How to Translate Names =====

There are many things to keep in mind when translating a name.
  * Proper names sometimes reveals information about the item.
  * Personal names may reveal information about the origin or history of the family.
  * Personal names may reveal information about the characteristic of the person.

There are several ways to handle translating a name:
  * Do not change the name from the Source text in the translated text. 
  * The name is changed to conform with the phonic or graphic rules of the translated text. (See [[en:ta:vocabulary:translatenames:transliterate]])
  * The source text name is replaced in the translated text by a name that has the same cultural meaning as the original name.

====== Check Notes Tags ======

==== Check on Bad Notes Tags ====

This query checks if the first tag is one of the valid tags required to make the system work correctly: {draft|check|review|text|discuss|inspect|fixed|publish}. Pages listed here are likely being overlooked.(EQ)

{{pagequery>@en:bible:notes \{\{tag>(?!draft|check|review|text|discuss|inspect|fixed|publish); fulltextregex; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;fontsize=90%}}

{{door43pages @:en:bible:notes -r -q="\{\{tag>(?![^}]*draft)(?![^}]*check)(?![^}]*review)(?![^}]*text)(?![^}]*discuss)(?![^}]*inspect)(?![^}]*fixed)(?![^}]*publish)[^}]*\}\}" -exclude:home -simpleLineBreak -title}}

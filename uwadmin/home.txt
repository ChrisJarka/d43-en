====== This is the unfoldingWord Administration section ======

Individual administrative pages exist under the project code namespace, e.g. **[[:en:uwadmin:en:obs:status|en/obs/status]]**.  When new languages are setup the status page for that language is automatically created.


See [[en:uwadmin:weeklychanges|Weekly Changes]] in OBS by namespace.

See [[en:uwadmin:monthlychanges|Monthly Changes]] in OBS by namespace.

To see which languages have been published to unfoldingWord, see [[en:uwadmin:pub_status|unfoldingWord OBS Published Languages]].

===== Select an Input Method =====

Complete these steps to select your input language:
  - Tap **Settings**.
  - Tap the **Input Language**  button.
  - At the list of Input languages, use a tap-swipe-up action to see all the languages on the list and locate the target language you want to use.
  - To select a language, tap the box to the right of the language.
  - Use the back arrow on the Android menu to go back to the **Setup**  window.

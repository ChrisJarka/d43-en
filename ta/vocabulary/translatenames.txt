===== How to Translate Names =====


There are many things to keep in mind when translating a name.


    * Proper names sometimes reveal information about the item.
    * Personal names may reveal information about the origin or history of the family.
    * Personal names may reveal information about the characteristic of the person.


There are several ways to handle translating a name:


    * Do not change the name from the Source text in the translated text.
    * Change then name to conform with the phonic or graphic rules of the translated text. (See [[:en:ta:vocabulary:translatenames:transliterate|Transliterate]])
    * Replace the source text name with a name that has the same cultural meaning as the original name.
    * <font 13px/arial;;#000000;;#ffffff>Pronounce the name as it is pronounced in the source language or another language, and using your language's spelling system, write it in a way that people will read it with that pronunciation. (See Transliterate)</font>

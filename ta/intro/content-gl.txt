===== Everything That Needs To Be Translated for Gateway Languages =====

If your language is a Gateway Language, then your language will serve as a source text for other languages to translate. As a result, all of the translation resources need to be translated so that speakers of these other languages can use them as they translate. Below is a list of everything that needs to be translated into each Gateway Language, along with the checking level that it needs to acheive before it can be used as a source text for another language.

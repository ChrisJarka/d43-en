====== Standard Notes for Jeremiah ======

This is a growing list of 'standard notes' that occur frequently in Jeremiah. Please use these so that there are consistent notes about the same topic throughout the book.

When a standard note includes brackets (see examples below), replace the brackets with the detail that is appropriate for the text you are working on.

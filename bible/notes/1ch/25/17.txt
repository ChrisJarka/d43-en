===== translationNotes: =====

  * **General Information:** - This continues the list begun in [[:en:bible:notes:1ch:25:09|1 Chronicles 25:9]].

  * **tenth ... thirteenth** - "the 10th ... the 11th ... the 12th ... the 13th" or "number 10 ... number 11 ... number 12 ... number 13." This shows the order in which the families were chosen. (See: [[:en:ta:vol2:translate:translate_ordinal]] and [[:en:ta:vol2:translate:translate_numbers]])
  * **Shimei** - Translate this man's name the same as you did in  [[:en:bible:notes:1ch:25:01|1 Chronicles 25:3]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **twelve persons** - "12 people" (See: [[:en:ta:vol2:translate:translate_numbers]])
  * **Azarel** - This is another spelling for "Uzziel" in [[:en:bible:notes:1ch:25:04|1 Chronicles 25:4]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **Hashabiah** - Translate this man's name the same as you did in [[:en:bible:notes:1ch:25:01|1 Chronicles 25:3]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **Shubael** - Translate this man's name the same as you did in [[:en:bible:notes:1ch:25:04|1 Chronicles 25:4]]. (See: [[:en:ta:vol1:translate:translate_names]])

====== Translate Names ======

===== Description =====

In English there are direct and non-direct (indirect) quotations.


===== Template =====

If there are many new names in a chunk that are not listed in the Important Terms, simply write the following at the top of the translation notes: \\ 
See: Translate Names

If this is the first time a person or place is named in the  book and it is not an important term, write a note like this:

  - **(Required)**  Write the name  from the ULB in bold
  - (Optional) Give a clarifying translation
  - (Optional)  Identify what it is a name for (place, man, woman, spirit, god, etc.) or give information about it.
  - (Optional) Give a translation alternative
  - (Optional)  Add TA Link: (See: Translate Names)
  * **(1) ULB text**  - **(3)**  Identify or give info **(5)**  (See: Translate Names)

If this is a name that has been not been used in recent chapters or chunks, write a note like this: 

  - **(Required)**  Write the name or list the names from the ULB in bold
  - (Optional)  Give a clarifying translation
  - (Optional)  Identify who or what it refers to
  - (Optional) Give a translation alternative
  - **(Required)** Refer translators to where they wrote it in an earlier chunk.

  * **(1) ULB text**  - **(3)**  Identify. **(5)**  See how you wrote this in )
===== Bible Examples =====

Field reference numbers are included for clarity purposes only. They are omitted in the actual note.

=== Gen 25:9-10  ===

Isaac and Ishmael, his sons, buried him in the cave of Machpelah, in the field of Ephron the son of Zohar the Hittite, which is near Mamre. This field Abraham had bought from the children of Heth. Abraham was buried there with Sarah his wife. 

  * **(1) Machpelah** - **(3,5)** See how you translated this place name in 23:9.
  * **(1) Ephron…Zohar** - **(3,5)** See how you translated these men's names in 23:8.
  * **(1) Mamre** - **(3,5)** See how you translated this place name in 23:17.
  * **(1) children of Heth** - **(2)** “the descendants of Heth” or “the Hittites.” **(5)** See how you translated “sons of Heth” in 23:5.

====== Check OBE Tags ======

==== Check on Bad OBE Tags ====

This query checks if the first tag is one of the valid tags required to make the system work correctly: 

{{pagequery>@:en:obe \{\{tag>(?!draft|create|delete|check|review|text|discuss|publish); fulltextregex; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}


==== Check for Well-Formed OBE Tags ====

The following query checks that every page that has curly braces"{}" include the key word "tag>".

{{pagequery>ns:en:obe -en:obe:home -en:obe:kt:home - en:obe:other:home "{{" -"{{tag>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}

regex

{{pagequery>@:en:obe \{\{tag>(?!\{\{tag>); fulltextregex; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}

====== OBE: Templates ======

==== OBE Term Template ====

The code for creating an OBE Key Term link is:

<code>
[[:en:obe:kt:term|term title]]
</code>

The code for creating an OBE Other Term link is:

<code>
[[:en:obe:other:term|term title]]
</code>

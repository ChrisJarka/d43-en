===== translationNotes: =====
  * **General Information:** - This continues the list begun in [[:en:bible:notes:1ch:25:09|1 Chronicles 25:9]].

  * **the fourteenth ... the seventeenth** - "the 14th ... the 15th ... the 16th ... the 17th" or "number 14 ... number 15 ... number 16 ... number 17." This shows the order in which the families were chosen.
 (See: [[:en:ta:vol2:translate:translate_ordinal]] and [[:en:ta:vol2:translate:translate_numbers]])
  * **Mattithiah** - Translate this man's name the same as you did in [[:en:bible:notes:1ch:25:01|1 Chronicles 25:3]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **twelve persons** - "12 people" (See: [[:en:ta:vol2:translate:translate_numbers]])
  * **Jeremoth** - This is another spelling for "Uzziel" in [[:en:bible:notes:1ch:25:04|1 Chronicles 25:4]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **Hananiah** - Translate this man's name the same as you did in  [[:en:bible:notes:1ch:25:04|1 Chronicles 25:4]]. (See: [[:en:ta:vol1:translate:translate_names]])
  * **Joshbekashah** - Translate this man's name the same as you did in  [[:en:bible:notes:1ch:25:04|1 Chronicles 25:4]]. (See: [[:en:ta:vol1:translate:translate_names]])

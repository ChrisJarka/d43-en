===== Setting Up translationStudio-Desktop =====

This module answers the question: //How do I set up translationStudio-Desktop?//

Before you start this module have you learned about:
  * //[[en:ta:vol1:process:platforms|Selecting a Platform]]//

==== Installing the Program ====

translationStudio-Desktop can be used on Windows, Mac, and Linux. 

To install the program, go to https://unfoldingword.org/translationstudio/ . Navigate to the "Desktop" section. Carefully follow the instructions for installing this program. 

==== Using the program ====

Once installed, translationStudio-Desktop is designed to work very similarly to translationStudio-Android.

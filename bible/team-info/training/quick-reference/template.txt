====== General Template ======


===== Description =====

The quick Reference template uses the following format for creating notes. Which fields are actually included in a note depends on the type of note being created.

===== Template =====

Create notes as follows:

  - (Required) State the topic phrase or the phrase that includes the topic word
  - (Optional) State a clarifying translation
    - mainly used with figures of speech
  - (Optional) State the meaning of the topic.
    - used to explain figures of speech
    - used to explain meanings
  - (Optional) State an alternate translation
    - used to state translation suggestions for Figures of Speech and Grammar topics 
  - Add TA Link

  * **topic phrase** - clarifying translation.  Text meaning. Translation suggestion. (See: Translation Academy Topic)

==== Notes ====

  * All Notes require Topic Phrase (1).
  * The kind of note written determines which optional fields must be included.


===== Bible Examples =====

=== Luk:24:16 ===

"But their eyes were prevented from recognizing him."

  * **Their eyes were prevented from recognizing him**  - "Their eyes were kept from recognizing Jesus." The men's ability to recognize Jesus is spoken of as their eyes' ability to recognize him. It can also be translated as "They were kept from recognizing him" or "Something prevented them so they could not recognize him." (See: [[:en:ta:translate:synecdoche]])

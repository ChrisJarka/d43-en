====== Standard Notes for Judges ======

This is a growing list of 'standard notes' that occur frequently in Judges. Please use these so that there are consistent notes about the same topic throughout the book.


SAMPLE
**the Chebar canal** - See how you translated this in 1:1, 3. 

<code>
  * **the Chebar canal**  - See how you translated this in [[en:bible:notes:ezk:01:01|1:1, 3]].
</code>

==== Open the Settings Menu ====

  * Click on the 3 dot menu from either the 'Your Translation Projects' or the 'Project Editing' windows.
  * Choose 'Settings' on the menu.

Use the back arrow at the top left of the screen to leave Settings.

==== General Section ====

=== GL Mode ===

**Display additional translation options for gateway languages**
  *Click on the empty square to allow this option.
Note: A blue square with a white check mark means that the option has already been chosen.

The decision to turn on the GL mode needs to be done before a project has been started. This decision needs to be made because of how the project file is named when saved. Please make sure that all your translators are creating projects from the same mode (GL on or off) if they want to combine the projects. 

=== Target Translation Font ===

The name of the chosen font is displayed. Click on the words 'Target Translation Font' to change the displayed font in the text.
  * Scroll through the names of the fonts and click on the name to select the desired font to be displayed in the Target language text.
  * Select the font and click on 'Confirm' to save.

=== Source Translation Font ===

The name of the chosen font is displayed. Click on the words 'Source Translation Font' to change the displayed font in the text.
  * Scroll through the names of the fonts and click on the name to select the desired font to be displayed in the Source text.
  * Select the font and click on 'Confirm' to save.

=== Font Size ===

The size of the chosen font is displayed. Click on the words 'Font size' to change the displayed font size in the text.
  * Choose from 'small', 'normal', and 'large' font sizes by clicking on the size to select.
  * Select the size and click on 'Confirm' to save. Or click on 'Cancel' to not change the font size.

=== Backup Location ===

The location of the backup files is displayed here. You may change the file location by:
  * Click on the section to open the Computer's File Manager.
  * Navigate to another file folder on the computer's hard drive.
  * Click on the File Manager's file folder.
  * Click the 'Select Folder' button. The window closes and the changes are saved.

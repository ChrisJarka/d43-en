===== Getting Started =====

This module answers the question: What do I need to have in place before getting started?\\
Before you start this module have you learned about:
  * [[en:ta:vol1:process:process_manual|Introduction to the Process Manual]]

Before getting started, you must be able to:

   * Agree with the [[en:ta:vol1:intro:statement_of_faith|Statement of Faith]], [[en:ta:vol1:intro:translation_guidelines|Translation Guidelines]], and the [[en:ta:vol1:intro:open_license|Open License]]. (Everyone involved in the translation and checking will need to [[https://unfoldingword.org/assets/docs/legal/unfoldingWord%20-%20Guidelines%20&%20License.pdf|sign a statement]] that work is being done according to those standards.)
   * Have command of a source language into which the text and translation helps have been translated and adequately checked.
   * Gain access to the source text and translation helps either in print form, or online at https://door43.org, or through translationStudio - our mobile app platform (see https://unfoldingword.org/translationstudio/)
   * Have access to an internet-connected computer or tablet where the translation can be uploaded and audio (if desired) can be recorded. See "[[en:ta:vol1:tech:hardware|Suggested Hardware]]"


You should be able to: 

   * Know where to get answers to my questions. See "[[en:ta:vol1:intro:finding_answers|Get answers to my questions]]"




Next we recommend you learn about:

===== Language Information =====

This module answers the question: How do I find language information?\\
Before you start this module have you learned about:
  * [[en:ta:vol1:intro:uw_intro|The unfoldingWord Project]]

==== Finding My Language ===

For each resource, there is a dropdown in the top right-hand corner. 
  * Tap on the dropdown menu

All the languages that have published content for that resource will appear. 

Note: As translations are published, you must manually [[en:ta:vol1:tech:uW_update_content|update]] the app to get that new content.
